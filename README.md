# SecureVault
SecureVault — менеджер паролей
Создано веб-приложение SecureVault.html с такими возможностями:

# Local storage
## Шифрование
AES-256-GCM для шифрования данных
PBKDF2 с 600 000 итераций (NIST, OWASP)
Ключ выводится только локально, данные не отправляются
## Функции
Генератор паролей — длина 12–64, опции A–Z, a–z, 0–9, спецсимволы, crypto.getRandomValues
Экспорт/импорт для Яндекс Браузера — CSV в формате url, username, password, name, extra
Импорт/экспорт JSON — для резервных копий
Категории — фильтрация по категориям
Поиск — по названию, логину и категории
## Интерфейс
Тёмная тема
Шрифты Space Grotesk и JetBrains Mono
Минималистичный дизайн
Защищённое от XSS экранирование вывода
## Безопасность
Пароли не хранятся в DOM (поиск по id в памяти)
Пароли закрыты при вводе
Предупреждение при экспорте (данные в CSV/JSON в открытом виде)
Запуск: откройте index.html в браузере. Хранилище сохраняется в localStorage и шифруется мастер-паролем.


pass:12345

# File storage


Изменения
Файловое хранилище
Загрузить файл — выбор .securevault или .json, ввод мастер-пароля, расшифровка и работа с записями
Создать новое — пустое хранилище с мастер-паролем
Сохранить — сохранение зашифрованного файла на диск
Загрузить — смена файла хранилища (другой файл, можно другой мастер-пароль)
## Работа
Открыть index.html в любом браузере
Загрузить свой .securevault или создать новое хранилище
Добавлять, редактировать, удалять записи
Нажать Сохранить, чтобы скачать актуальный файл
## Переносимость
Файл можно копировать на другой компьютер
Использовать в любом браузере (Chrome, Firefox, Safari, Edge и др.)
Хранить на флешке, в облаке, на другом диске
## Миграция
Если в localStorage остались старые данные, появится кнопка «Перенести из браузера». Введите мастер-пароль, нажмите её, затем Сохранить, чтобы сохранить всё в файл.
